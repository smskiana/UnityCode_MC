功能特性
1. 区块化世界管理 
 1) 异步加载体素区块
 2) 支持动态生成与卸载
 3) 使用 Addressables 管理资源，提高加载性能
2. 角色控制
1) 自研栈式状态机，支持状态压栈/弹栈
2) 支持移动、跳跃、攻击、受击等行为
3) 支持状态中断与默认迁移逻辑
3. 背包与物品系统
1) 支持物品堆叠、拆分、合并
2) 可动态调整背包容量
3) 事件驱动 UI 实时刷新
4. 数值与效果系统

1) 实现效果驱动的属性变化（Buff/Debuff）
2) 支持伤害修正与属性叠加
3) 每帧安全计算数值，支持扩展效果类型
5. 模块化架构
1) 数据与逻辑分离，ScriptableObject 驱动全局配置
2) 接口解耦（IControl、IEffectTarget、IDamageSource）
3) 易于扩展与多人协作开发
